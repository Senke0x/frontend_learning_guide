# Day 4 - 作业目录

Day 4 课程的实践作业目录，包含三个独立的作业项目。

## 目录结构

```
day4/
├── homework1-codegen-demo/      # 作业1：代码生成演示
├── homework2-api-mock/          # 作业2：API Mock 实现
└── homework3-airbnb-search/     # 作业3：Airbnb 搜索功能
```

## 作业说明

### 作业1: codegen-demo

代码生成演示项目，学习如何使用代码生成工具自动生成类型定义和 API 客户端。

**学习目标**:
- 理解代码生成的概念和应用场景
- 使用 OpenAPI/Swagger 生成 TypeScript 类型
- 自动化 API 客户端代码生成

### 作业2: api-mock

API Mock 实现项目，学习如何创建模拟 API 服务用于前端开发和测试。

**学习目标**:
- 理解 API Mock 的作用和使用场景
- 使用 Mock 工具创建模拟数据
- 前后端分离开发实践

### 作业3: airbnb-search

Airbnb 搜索功能实现，综合运用所学知识构建一个搜索功能。

**学习目标**:
- 实现搜索功能的前端交互
- 处理搜索请求和结果展示
- 优化用户体验（防抖、加载状态等）

## 使用说明

1. 进入对应的作业目录
2. 按照作业要求完成实现
3. 可参考 `/answers` 页面查看参考答案

## 提交作业

完成作业后，可以：
1. 在本地运行测试验证功能
2. 对比参考答案检查实现
3. 提交代码进行 review
